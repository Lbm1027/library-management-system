Main features

The Books, Categories, and Users pages are the primary focus of this demo. Each of these pages supports full Create, Read (from the database), Update (user updates depend on membership type), and Delete operations.

Tip: Reservations and borrow records are searched using a user ID. Visit the Users page first to find the ID you need, then run searches on the Reservations or Borrow Records pages.

Other pages

The remaining pages—Reservations, Borrow Records, and Fines—are included for fun demonstrations. Some results (for example, reservation and record searches) are fixed in the database. To change them, update the underlying data directly.
